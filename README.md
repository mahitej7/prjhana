# Property Of Hafsa

Static website ready for GitHub + Vercel deployment.

## Run locally

```bash
./run-mobile.sh
```

Open the printed `http://<wifi-ip>:8010` URL from another device on the same Wi-Fi.

## Deploy on Vercel

Import the GitHub repository in Vercel and keep the default static-site settings:

- Framework Preset: Other
- Build Command: none
- Output Directory: `.`
